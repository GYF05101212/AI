# HW05 报告

## 一、任务一：极简CNN学习摘要

### 1.1 文章学习要点

文章介绍了使用 PyTorch 实现一个极简 CNN 识别 MNIST 手写数字。核心内容包括：

- **CNN 结构**：一个卷积层（16个3×3卷积核）→ ReLU → 最大池化（2×2）→ 全连接层（10个输出）
- **参数量**：约 31,530
- **训练结果**：5个 epoch 后测试准确率约 98.14%

### 1.2 CNN 核心组件理解

| 组件 | 作用 |
|------|------|
| 卷积层 | 提取图像局部特征（边缘、纹理等） |
| ReLU 激活 | 引入非线性，增强表达能力 |
| 池化层 | 降维，减少计算量，提取主要特征 |
| 全连接层 | 将提取的特征映射到分类结果 |

### 1.3 运行结果

- 测试准确率：98.08%
- 训练耗时：150 秒
- 模型参数量：31,530

## 二、任务二：LeNet-5 实现

### 2.1 网络结构

| 层 | 类型 | 核大小/步长 | 输入尺寸 | 输出尺寸 | 参数量 |
|---|------|------------|---------|---------|--------|
| C1 | 卷积层 | 5×5 / 1 | 32×32×1 | 28×28×6 | 156 |
| S2 | 平均池化 | 2×2 / 2 | 28×28×6 | 14×14×6 | 12 |
| C3 | 卷积层 | 5×5 / 1 | 14×14×6 | 10×10×16 | 1,516 |
| S4 | 平均池化 | 2×2 / 2 | 10×10×16 | 5×5×16 | 32 |
| C5 | 卷积层 | 5×5 / 1 | 5×5×16 | 1×1×120 | 48,120 |
| F6 | 全连接层 | - | 120 | 84 | 10,164 |
| 输出 | 全连接层 | - | 84 | 10 | 850 |
| **总计** | | | | | **60,850** |

### 2.2 训练配置

| 参数 | 值 |
|------|-----|
| 优化器 | SGD（lr=0.01，momentum=0.9） |
| 损失函数 | 交叉熵损失 |
| 批量大小 | 64 |
| 训练轮数 | 10 |
| 设备 | CPU / GPU |

### 2.3 运行结果

- 测试准确率：99.04%
- 训练耗时：4.69 分钟
- 模型参数量：91,946

### 分析
